# Earthman64

A Commodore 64 tribute to my father.


## Building
Requires [dasm](https://dasm-assembler.github.io/).
```
dask main.asm -oearthman.prg
```
Run with [VICE](https://vice-emu.sourceforge.io/).

After loading, enter `SYS 2064` to run.
